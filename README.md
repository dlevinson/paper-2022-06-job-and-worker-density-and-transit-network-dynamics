## What Is Included

This package stages the public MetCouncil/Metro Transit stop boarding and alighting source workbook set supplied for this paper. The paper's overlapping analysis window is 2013-2017, and the adjacent 2018 workbook is retained with the rest of the public source workbooks so the local source set stays together. The original XLSX files are preserved in `data/metc_stop_boardings_alightings/original_xlsx/`, and CSV sidecars are provided in `data/metc_stop_boardings_alightings/csv/` for easier inspection. The source metadata from MetCouncil is preserved in `data/metc_stop_boardings_alightings/metadata/`.

The folder also contains `metadata/DATA_DICTIONARY_AUTOGENERATED.csv`, `metadata/METC_STOP_BOARDINGS_ALIGHTINGS_WORKBOOK_SUMMARY.csv`, `metadata/METC_STOP_BOARDINGS_ALIGHTINGS_FILE_MANIFEST.csv`, `metadata/SOURCE_BOUNDARY_DECISIONS.csv`, and `metadata/PUBLIC_SOURCE_LINKS.csv`.

## Exclusions And Boundary Decisions

LEHD/LODES files and the MetCouncil transit stop dataset are not duplicated locally because they are public source inputs and are documented by stable source pointers. The macOS `.DS_Store` file from the source folder is excluded.

## Limitations

No local analysis scripts were identified for this package. This is a source-data package with public-source links and modern CSV sidecars, not a turnkey replication workflow. The local paper PDF in `paper/` is included for audit convenience; publisher redistribution rights should still be reviewed before any public GitHub release that includes the PDF itself.
